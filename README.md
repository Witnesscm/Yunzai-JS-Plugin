# Yunzai-plugin

自用云崽bot js插件

## 食用方法

- js文件放入Miao-Yunzai/plugins/example目录下
- html文件夹放入Miao-Yunzai/resources目录下

## 插件列表

| 插件           | 命令             | 功能                                                         |
| -------------- | --------------- | ------------------------------------------------------------ |
| 体力免验证码.js | `#体力 *体力`    | 从米游社小组件获取体力详情，需要安装cvs插件绑定stoken后使用       |
